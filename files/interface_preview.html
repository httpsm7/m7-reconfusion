🔬 Scanning Phases
Phase 0
🔐 Scope Validation
auth-check scope-log dir-setup
Mandatory authorization confirmation, timestamp logging, and project folder structure creation before any scanning begins.
Phase 1
🌐 Subdomain Enumeration
assetfinder subfinder amass
Parallel subdomain discovery using 3 tools. Results are merged, deduplicated, and normalized automatically.
Phase 2
🌍 Live Host Detection
httpx status-codes headers
Probe all subdomains for live HTTP/HTTPS hosts. Captures status codes, server headers, content-type, and page titles.
Phase 3
🔎 Port Scanning
naabu nmap services
Fast port scanning with naabu (primary) or nmap (fallback). Identifies open ports, services, and IP addresses.
Phase 4
🧪 Advanced Scanning
katana dalfox nuclei
URL crawling → XSS testing → Template-based CVE scanning. Full pipeline for comprehensive vulnerability discovery.
Phase 5 + 6
📊 Analysis + Report
parser jinja2 bootstrap
Intelligent deduplication, severity grouping, and a beautiful dark-themed HTML report with collapsible findings.
